\subsection{Gebräuchliche griechische Buchstaben}
In er Informatik ist es üblich griechische Buchstaben zu
verwenden. Ein Grund hierfür ist, dass es so möglich wird mit einer
größeren Anzahl von Unbekannten arbeiten zu können, ohne
unübersichtliche und oft unhandliche Indices benutzen zu müssen.
\index{griechische Buchstaben}\index{Buchstaben!griechische}

\bigskip

\noindent Griechische Kleinbuchstaben:
\begin{itemize}
%
\item $\alpha$ (Alpha)
%
\item $\beta$ (Beta)
%
\item $\gamma$ (Gamma)
%
\item $\delta$ (Delta)
%
\item $\phi$ (Phi)
%
\item $\varphi$ (Phi)
%
\item $\xi$ (Xi)
%
\item $\zeta$ (Zeta)
%
\item $\epsilon$ (Epsilon)
%
\item $\theta$ (Theta)
%
\item $\lambda$ (Lambda)
%
\item $\pi$ (Pi)
%
\item $\sigma$ (Sigma)
%
\end{itemize}
